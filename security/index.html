security

黑客工程化是很多做安全的追求。

黑客工程化的意思是： 我们所有的技能需要成体系活动，能交给机器的都交给机器，我们控制机器木偶们来加速提高我们的效率。为了最优化这个工程化，必须要明白什么技能是必须精通的，什么技能是够用就好的，什么技能是拿来主义。

什么黑客技术是最好的技术？ 黑产或APT 组织已经在实战的技术。
